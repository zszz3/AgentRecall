# 会话查找、同步与桌面体验升级

## 新增功能

- macOS 顶部菜单栏新增轻量会话搜索，支持键盘快速选择并打开会话；“保留在程序坞”可在设置中关闭，默认开启。
- 本地会话中的图片和文件附件支持预览或用系统应用打开；远程会话同步默认同时上传可用附件，并支持按需查看。
- 应用更新会显示真实下载进度，以及校验、安装、验证和重启阶段；新版本验证通过后才替换当前版本。
- README 直接提供国内 npm 与 Electron 镜像安装命令。

## Bug 修复

- Cursor Remote SSH 会话现在会按实际远程环境归类，不再错误显示为本地会话；空的 Cursor 会话记录也不会再出现在列表中。
- Codex 额度查询遇到临时网络错误时会自动重试；刷新仍失败时保留同一账号最近一次成功数据，重新登录或切换账号后自动刷新。
