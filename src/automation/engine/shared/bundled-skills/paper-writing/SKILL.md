# 论文写作

## 工作方式

此技能用于学术论文写作辅助：选题框定、结构设计、论证组织、证据整理、引用和修改。它只提供写作辅助，不伪造研究过程或学术结果。

1. 先确认论文类型：课程论文、毕业论文、会议论文、期刊论文、开题报告、文献综述或研究计划。
2. 明确研究问题、学科背景、目标读者、字数要求、引用格式、截止时间、已有材料、数据来源和老师或期刊的格式要求。
3. 如果用户只有主题，先帮助收敛研究问题：说明问题价值、研究边界、核心概念、可行方法和可能创新点。
4. 如果用户已有草稿，优先做结构诊断：论点是否清晰、证据是否支撑、段落是否连贯、方法是否可复现、结论是否越界。
5. 协助输出摘要、引言、相关工作、方法、实验或案例分析、讨论、结论和修改建议；每段要服务于主论点。

## 输出格式

- 研究问题和论文主张：用简短句子明确论文要回答什么。
- 大纲：按章节列出核心论点、证据需求和写作重点。
- 正文草稿或改写建议：保持学术表达，避免口语化和空泛判断。
- 文献与证据清单：区分已确认来源、用户需补充来源、待核验来源。
- 修改清单：按结构、论证、语言、格式、引用分组。

## 学术诚信

不要伪造文献、DOI、作者、数据、实验结果或引用页码。不了解来源时必须标注“待核验”。不要替用户规避学校或期刊规则；需要引用、查重、伦理审批或数据授权时直接提醒。
