# 最终审查清单

1. 逐条核对简历中的姓名、联系方式、组织、岗位、日期、技能、数字和成果是否有来源证据。
2. 检查是否把 JD 关键词误写成候选人能力，或出现关键词堆砌、重复 bullet 和空泛形容词。
3. 检查高优先级岗位要求是否在 match-report.md 中有证据或明确的未覆盖标记。
4. 检查三份文件内容一致，且 match-report.md 不包含不可解释的 ATS 百分比分数。
5. 检查 tailored-resume.html 中没有 `{{...}}` 占位、示例文字、空章节、脚本或外部资源依赖。
6. 检查 HTML 为单栏打印布局，联系方式可复制，链接包含可读文本，页面没有横向溢出。
7. 检查 tailored-resume.md 可独立阅读和继续编辑，标题层级、列表和日期格式一致。
8. 若发现任何无证据事实，删除该内容并在 match-report.md 中记录为未覆盖或待确认。
